ğŸ¯ Quick Command Cheat Sheet

Check Status

git branch              # Which branch am I on?
git status              # Do I have changes?
git remote -v           # Where am I pushing to?

Save Work

git add .               # Stage all changes
git commit -m "msg"     # Save changes
git stash save "msg"    # Temporarily save (if uncommitted)

Switch Branches

git checkout feature/my-branch       # Go to feature branch
git checkout -b feature/new-branch   # Create and go to new branch

Sync with Team

git pull origin main              # Get latest from main
git merge main                    # Bring main into your branch
git push origin feature/my-branch # Send your work to GitHub

Get Work Back

git stash pop           # Get stashed work back
git stash list          # See all stashed work
